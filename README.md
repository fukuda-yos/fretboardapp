# Fretboard Pitch Map

ギター/ベースのフレットボード可視化・コード理論学習・練習支援のWebアプリです。
単一の HTML ファイル（`index.html`）で完結しており、ブラウザで直接開いて使えます。

## 主な機能

- フレットボード表示（ギター 24F / ベース 20F、音高の色分け、弦の上下反転）
- クリックで発音 + 同音ハイライト
- コードハイライト / キー & ダイアトニックコードパネル / コード直接入力
- コード試聴（同時発音 + アルペジオ6種、ボイスリーディング対応）
- スケール表示（12種、コードと同時表示可）
- メトロノーム & リズムパターン（rock8 / pops16 / rockabilly / shuffle / bossa ほか）

## 使い方

`index.html` をブラウザで開くだけです。ビルドツール・外部ライブラリは不要です。
